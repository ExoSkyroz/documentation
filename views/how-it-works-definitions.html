How do definitions work?


Definition serves for setting of framework.

  • Definition is plain JavaScript file.
  • Framework automatically loads each definition.
  • You can create unlimited count of definitions.
  • Definition can access to framework functions, properties and events.
  • Definition can inject controller, config and module from URL.
  • Definition can create any files.
  • Definition can inject from URL address - Example.
