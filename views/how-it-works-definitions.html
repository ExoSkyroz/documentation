How definitions work?


Definitions initialize framework settings.

  • A Definition is plain JavaScript file.
  • Framework automatically loads each definition.
  • You can create unlimited definitions.
  • Definitions can access framework functions, properties and events.
  • Definitions can inject controller, config and module from URL.
  • Definitions can create files.
  • Definitions can be injected from a URL address - Example.
