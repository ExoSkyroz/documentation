@{meta('Total.js command-line tool')}

Command-line tool


Firstly you must install total.js from NPM as global package

$ npm install -g total.js

Arguments


Command-line help:

$ total --help

@{image('cmd.png', 780, 480, 'total.js arguments')}

Example: create e.g. Angular.js project


$ mkdir angular
$ cd angular
$ total -angular

@{image('cmd-angular.png', 780, 480, 'total.js creates angular.js application project')}

Run application:

$ node debug.js
