How do models work?


Models serve for database schemas. You can use models in other projects.

  • Model is plain JavaScript file.
  • All models are stored in your-app/models/ directory.
  • You can create unlimited count of models.
  • Model can inject from URL address - Example.
Model example

Model definition


// Model name: article
exports.comment = function() {
	return { content: String, date: Date };
};

Usage

Universal usage

var model = framework.model('article').comment();

In controllers

function view_homepage() {
	var self = this;

	// GETTING THE MODEL
	var model = self.model('article').comment();

	self.empty();
}

Monitoring of models


total.js supports monitoring of models via model usage. Usage example. 

var counter = 0;

exports.usage = function() {
	return { counter: counter };
};

exports.user = function() {
	counter++;
	return { alias: '' age: 0 };
};
