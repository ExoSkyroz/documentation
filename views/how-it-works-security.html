How does security work?



  • Framework automatically protects XSS (Cross-site scripting).
  • For protecting of CSRF attacks you must use XHR sending of forms.
  • Views and Templates render encoded HTML as default.
  • Framework supports method for settings CORS.
  • Framework supports auto-binding model into forms.
